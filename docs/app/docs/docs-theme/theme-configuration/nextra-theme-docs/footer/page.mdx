# `Footer`

The footer area of the website. You can specify content for your default footer.

<Table>

|           |                  |                                                                                           |
| --------- | ---------------- | ----------------------------------------------------------------------------------------- |
| children  | `ReactNode{:ts}` | Content of the footer.<br/>Default: `"MIT ${new Date().getFullYear()} © Nextra."{:json}` |
| className | `string{:ts}`    | CSS class name                                                                            |

</Table>

## Example

You can add content, such as copyright information by passing it as `children`
of the `Footer` component:

```jsx filename="app/layout.jsx"
import { Footer, Layout } from 'nextra-theme-docs'

export default function MyLayout({ children, ...props }) {
  return (
    <Layout>
      {children}
      <Footer>
        MIT {new Date().getFullYear()} ©{' '}
        <a href="https://nextra.site" target="_blank">
          Nextra
        </a>
        .
      </Footer>
      {children}
    </Layout>
  )
}
```

### Toggle Visibility

You can toggle visibility of the `<Footer>` on the specific pages by setting
`theme.footer` property in the `_meta.js` file:

```js filename="_meta.js"
export default {
  'my-page': {
    theme: {
      footer: false // Hide footer on this page
    }
  }
}
```
