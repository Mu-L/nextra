# `Footer`

The footer area of the website. You can either specify some content for the
default footer, or fully customize it with a custom component.

<Table>

|           |                  |                       |
| --------- | ---------------- | --------------------- |
| children  | `ReactNode{:ts}` | Content of the footer.<br/>Default: `"MIT ${new Date().getFullYear()} © Nextra."{:json}` |
| className | `string{:ts}`    | CSS class name        |

</Table>

## Copyright Information

You can add some simple content, such as copyright information to the default
footer:

```jsx filename="app/layout.jsx"
import { Layout, Footer } from 'nextra-theme-docs'

export default function MyLayout({ children, ...props }) {
  return (
    <Layout>
{children}
      <Footer>
        MIT {new Date().getFullYear()} ©{' '}
        <a href="https://nextra.site" target="_blank">
          Nextra
        </a>
        .
      </Footer>
      {children}
    </Layout>
  )
}
```
